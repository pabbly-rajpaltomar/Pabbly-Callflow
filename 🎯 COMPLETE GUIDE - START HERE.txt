PROBLEM: "localhost refused to connect"
SOLUTION: Run FINAL_FIX.bat to start servers


PROBLEM: "Login failed"
SOLUTION:
- Check email/password correct
- Check backend server is running
- Check database connection


PROBLEM: "Can't upload recording"
SOLUTION:
- Make sure file is audio format (MP3, M4A, etc.)
- File size should be under 50MB
- Backend server must be running


PROBLEM: "Recording not playing"
SOLUTION:
- Check backend server is running
- Refresh browser page
- Check browser console for errors


PROBLEM: "Database connection failed"
SOLUTION:
- Check PostgreSQL is running
- Check password in backend\.env is correct
- Run setup-db.js again


PROBLEM: "Changes not showing"
SOLUTION:
- Refresh browser (Ctrl+F5)
- Check if you saved files
- Restart FINAL_FIX.bat
