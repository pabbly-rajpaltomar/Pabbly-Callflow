============================================

When you change PC or want code on another computer:

1. Install these first:
   - Node.js: https://nodejs.org
   - PostgreSQL: https://www.postgresql.org/download/windows/
   - Git: https://git-scm.com/download/win

2. Open Command Prompt and run:
   git clone https://github.com/YOUR-USERNAME/Pabbly-Callflow.git

3. Go to the folder:
   cd Pabbly-Callflow

4. Setup database password:
   - Copy backend\.env.example to backend\.env
   - Open backend\.env in Notepad
   - Change "your_postgres_password_here" to your actual password

5. Start everything:
   Double-click: FINAL_FIX.bat

6. Done! Working on new PC! ‚úì
